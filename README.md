# Spelling Site

A simple spelling practice web app with pages for home, login, dashboard, quiz, and word list.

## Structure
- index.html: landing page
- login.html: authentication page
- dashboard.html: progress view
- quiz.html: spelling quiz
- wordlist.html: searchable word browser
- css/: styling files
- js/: JavaScript logic
- data/words.json: sample word data
